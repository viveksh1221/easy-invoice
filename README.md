# 🧾 EasyInvoice - Freelance Invoice Generator

EasyInvoice is a simple and efficient CLI-based invoice generator built using Python. It helps freelancers quickly create structured and professional invoices by adding client details, multiple services, and automatically calculating the total amount.

---

## 🚀 Features

- Add client and project details  
- Add multiple services  
- Automatic total calculation  
- Clean formatted invoice output  

---

```bash
python main.py
